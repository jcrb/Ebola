New Cases et Courbe pidmique

Depuis le 26/8, lOMS nindique plus le nombre de nouveaux cas. Il faut donc les calculer par soustraction des 2 derniers bilans.

Exemple avec la Guine:

source("ebola_functions.R")

a <- tapply(d$Total[d$Pays=="Guinea"], d$Date[d$Pays=="Guinea"], sum)
n <- length(a)
b <- a[n] - a[n-1]

Pour la Guine:
- dernier bilan: 2014-10-05
- nombre total de nouveau cas: 99

Il faut rpter lopration pour les autres comptes (Confirmed, Probable, suspected) et les autres pays => function

Il est possible de simplifier le calcul en utilisant linstruction diff qui calcule les diffrences successives.

state <- "Liberia"
a <- tapply(d$Total[d$Pays == state], d$Date[d$Pays == state], sum)
a1 <- names(a)
a2 <- as.numeric(a)
a3 <- c(0, diff(a)) # premier lment = 0 car 1re diffrence nulle
a3[a3 < 0] <- 0 # si diffrence ngative on ramne  0
b <- data.frame(a1,a2,a3)
names(b) <- c("date","cumul","delta")

# On cre autant d'enregistrements qu'il y a de cas
x.date <- rep(b$date, b$delta)
ebola <- epicurve.weeks(x.date, sunday = FALSE, axisnames = FALSE, col="#A8DDB5")
axis(1, at = ebola$xvals, labels = ebola$cweek, tick = FALSE, line = 0)
axis(1, at = ebola$xvals, labels = ebola$cmonth, tick = FALSE, line = 1)
title(main= paste0(state,": Ebola - Nouveaux cas"), ylab = "frquence", xlab = "2014 - semaines")

# voir la routine new.case()
xl <- new.case(d, state)
plot(as.Date(xl$date), xl$cumul, type="l")

plot(as.Date(xl$date), xl$delta, type="h")

barplot(xl$delta, names.arg=xl$date, las=2, cex.names = 0.8, main = paste0(state," - Nouveaux cas"))

# idem avec new.death()
xd <- new.death(d, state)
barplot(xd$delta, names.arg=xd$date, las=2, cex.names = 0.8, main = paste0(state," - Nouveaux Dcs"))

# taux de mprtalit brut et  15 jours
mb <- xd$cumul/xl$cumul
plot(mb, type="l")

xll <- c(1,1,xl$cumul) # impose un dcalage de 15 jours
xdd <- c(xd$cumul,1,1)
r <- round(xdd/xll, 2)
plot(r, type="l")
